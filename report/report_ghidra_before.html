1. Executive Summary

This binary is a Linux-based backdoor (Unknown Family) designed to provide remote access and execution capabilities to an attacker. The malware operates by establishing a raw socket interface, allowing it to intercept and craft custom UDP packets. It listens for incoming commands, decrypts them using a simple XOR cipher, executes them on the local system using a shell, and returns the output to the remote controller. The use of raw sockets (Protocol 0x11/UDP) and manual packet crafting suggests an attempt to bypass standard firewall inspections or blend in with normal network traffic. The threat level is High due to its ability to execute arbitrary commands and its stealthy networking approach.

  • Data Obfuscation: Implements a custom XOR encryption function (myEncryptFun) to hide command payloads and exfiltrated data.
  • Packet Crafting: Manually constructs network packets, calculating IP and UDP checksums to ensure validity.
  • Configuration: Reads configuration from a file named system.conf.
  • Stealth: Sets the "Don't Fragment" (DF) flag on outgoing packets to influence network handling.
