1. Executive Summary

This binary is a sophisticated Linux malware specimen that utilizes Google Sheets as a Command and Control (C2) infrastructure to evade detection via "Living off the Land" techniques.

Upon execution, the malware decrypts configuration data stored within its own binary using AES-128-CBC. It authenticates to Google Cloud Platform using OAuth2 with a JWT signed by an RSA key found in the decrypted configuration.

Once authenticated, it performs comprehensive system reconnaissance and enters a persistent polling loop, reading commands from the spreadsheet to execute arbitrary shell commands, exfiltrate files, or write files to the local disk.
