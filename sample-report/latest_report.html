1. Executive Summary

The analyzed binary 961ac6942c41c959be471bd7eea6e708f3222a8a607b51d59063d5c58c54a38d is a Linux x86 ELF malware identified as a Remote Access Trojan (RAT) with integrated proxy and port-forwarding capabilities. The malware utilizes statically-linked C++ libraries, evidenced by mangled class names such as cclient, cmy_socks_mgr (SOCKS proxy), and cport_fwd_mgr (port forwarding). It establishes communication with a Command and Control (C2) server using a custom binary protocol over TCP. The sample implements extensive cryptographic primitives including RSA (with PEM parsing supporting DES and AES encryption), MD5, SHA-1, and SHA-256 to secure communications and handle keys. It performs system reconnaissance by reading sensitive files like /etc/passwd and /proc/cpuinfo.
